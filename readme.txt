This plugin shows your websites google page rank

== Description ==

My first public plugin that shows your websites google pagerank in the dashboard. Can be disabled in the screen options menu and moved around at will.

Works 2.7+, I don't know if it works on anything below that.

Just really an example of how to put something into the dashboard.

== Installation ==

1. Upload the folder 'dashboard-google-pagerank' to your '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
